Class StackedLayout

A layout in which only one widget is visible at a time.

User code is responsible for managing the current layout index. The index defaults to -1, which means no widget will be shown. The index must be set to a valid index in order for a widget to be displayed.

If the current widget is removed, the current index is reset to -1.

This layout will typically be used in conjunction with another widget, such as a tab bar, which manipulates the layout index.
