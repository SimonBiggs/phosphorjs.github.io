Class Bootstrapper

A class which manages bootstrapping an application.

An application will typically define its own Bootstrapper subclass which overrides the necessary methods to customize the application.

Protected finalize

  • finalize(): void
    • Defined in src/shell/bootstrapper.ts:165

    Finalize the bootstrapping process.

    This is called after all plugins are resolved and intialized.

    It is the last method called in the bootstrapping process.

    This can be reimplemented by subclasses as needed.

    The default implementation attaches the shell widget to the DOM using the "main" element or document.body, and adds a window resize event handler which refits the shell on window resize.

    Returns void
