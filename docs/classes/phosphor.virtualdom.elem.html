Class Elem

A node in a virtual DOM hierarchy.

User code will not typically instantiate an elem directly. Instead, a factory function will be called to create the elem in a type-safe fashion. Factory functions for all standard DOM nodes are provided by the framework, and new factories may be created with the createFactory function.

An elem must be treated as immutable. Mutating the elem state will result in undefined rendering behavior.
