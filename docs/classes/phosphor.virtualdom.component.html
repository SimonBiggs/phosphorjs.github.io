Class Component<T>

A component which renders its content using the virtual DOM.

User code should subclass this class and reimplement the render method to generate the virtual DOM content for the component.

update

  • update(immediate?: boolean): void
    • Defined in src/virtualdom/component.ts:97

    Schedule an update for the component.

    This should be called whenever the internal state of the component has changed such that it requires the component to be re-rendered, or when external code determines the component should be refreshed.

    If the immediate flag is false (the default) the update will be scheduled for the next cycle of the event loop. If immediate is true, the component will be updated immediately. Multiple pending requests are collapsed into a single update.

    Parameters

    • Default value immediate: boolean = false

    Returns void
