Class LayoutSizer

A sizer object for the layoutCalc function.

Instances of this class are used internally by the panel layouts to implement their layout logic. User code will not typically use this class directly.

stretch: number

The stretch factor for the sizer.

This controls how much the sizer stretches relative to the other sizers when layout space is distributed. A sizer with a stretch factor of zero will only be resized after all stretch sizers and expansive sizers have been sized to their limits.
